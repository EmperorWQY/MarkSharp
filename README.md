# MarkSharp

#### Introduction
Parse Markdown to HTML for .NET

#### Usage  
[NuGet](https://www.nuget.org/packages/MarkSharp)

```
using MarkSharp;

var markdown = new Markdown("File Path");
var html = markdown.GetHtmlContent();
```
